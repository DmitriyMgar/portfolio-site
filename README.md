# dmgar.is-a.dev

Personal portfolio website — Head of AI / Fullstack Developer.

🔗 **Live:** [dmgar.is-a.dev](https://dmgar.is-a.dev)

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- TypeScript
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Framer Motion](https://motion.dev/) — animations
- [Radix UI](https://www.radix-ui.com/) + [shadcn/ui](https://ui.shadcn.com/) — components
- [Lucide](https://lucide.dev/) — icons

## Features

- Role switcher (Head of AI / Fullstack Developer) — adapts content to the selected role
- Bilingual (EN / RU)
- Animated space background
- Responsive design
